Overview

AskTabs lets you query and summarize the content of your open browser tabs using AI. This policy explains what data we access, how we use it, and how it’s protected.

What we access (temporarily)

Tab content
Text, titles, and URLs of currently open tabs — accessed only when you request a summary or ask a question.
Usage data (anonymous)
Aggregate, non-identifying telemetry such as feature usage counts to help improve the extension.

What we DO NOT collect

We do not collect or store:

  • Names, emails, addresses, or other personally identifiable info
  • Passwords, authentication tokens, or credentials
  • Financial data or payment details
  • Health records
  • Your emails, chats, or other personal messages
  • Precise GPS location or continuous activity (clicks/keystrokes)

How data is used

  • To generate answers and summaries from the content of open tabs.
  • To improve the extension using anonymous usage analytics.

Data handling & security

AskTabs processes tab content via a secure server endpoint when necessary. Tab content is processed temporarily and not stored permanently. We use encryption in transit and limit retained data to the minimum required for response generation.

Third-party services

AskTabs may use third-party services to deliver AI and hosting:

  • Google Gemini API — to generate AI responses.
  • Vercel / Hosting provider — to host secure API endpoints.

These services process data only as needed. Refer to their privacy policies for details.

Permissions

The extension requests the following Chrome permissions to function:

tabs, scripting, sidePanel, activeTab, storage, <all_urls>

Each permission is used solely for the purposes described above (reading open tabs, extracting content, displaying UI, and storing local settings/history).

Your rights

  • Disable or remove the extension anytime via Chrome settings.
  • Request deletion of any locally stored conversation history or settings.

Changes

We may update this policy occasionally. Changes will be posted here with an updated effective date.
